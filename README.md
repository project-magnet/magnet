## 사용기술
- Java17
- spring boot 3.2.0
- Spring Security 6.2.0
- Spring data JPA
- QeuryDSL
- JUnit5
- MySQL 8
- Docker
- gradle 8.5
- AWS EC2, cloudFront, route53
